# mister_ruka-palka
Example game on GiveAGame

# For run and build...
## Install libs 
**Write** in *your terminal*: `pip install giveagame` and `pip install pyinstaller`

## Build game
**Run** `build.bat`
*After running* in **project folder** *created new folder* with *name `dist`*

## Run game
*Run `dist/main.exe`*
